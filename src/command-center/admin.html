Invite New Team Member
Invitations must be sent from the server using the admin invite script. The browser cannot send invitations directly because it requires a privileged server key.

To invite a new team member, run this command on the server:
node src/command-center/invite.js "email@example.com" "Display Name" staff
Replace staff with manager if appropriate.

The invited person will receive an email with a link to set their password and log in.
Admin accounts cannot be created through the invite form. Contact your Supabase administrator to grant admin access.
